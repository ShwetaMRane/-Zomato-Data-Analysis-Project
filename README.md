🍽️ Zomato Data Analysis Project

This project focuses on analyzing restaurant data from Zomato to understand customer preferences, restaurant performance, pricing trends, and location-based patterns. Using Python and data visualization techniques, the analysis uncovers key factors influencing restaurant ratings and popularity.

Tech Stack:

Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

Key Insights:

Impact of online ordering and table booking on ratings
Most popular restaurant locations
Cost distribution across restaurants
Customer voting trends
Rating distribution analysis
